# Street Food Empire: Kingston Rush

**Street Food Empire: Kingston Rush** is a 2D mobile-first hybrid-casual street food tycoon game.

The player starts with a small Jamaican street food cart in **Kingston Night Market**, serves customers, earns coins, upgrades the stall, unlocks foods, hires staff, survives rush-hour events, and grows toward a global street food empire.

## Project status

Early MVP / prototype.

The current goal is to build a small, playable Phaser 3 prototype before adding polish, monetization, advanced content, or platform-specific features.

## Core gameplay loop

```text
Serve customers
→ Earn coins
→ Upgrade food, staff, and stall
→ Serve faster
→ Trigger rush hour
→ Unlock more content
→ Earn offline income
→ Return and progress further
